# Galaxy Strike

3D rail shooter, созданный в рамках курса «Complete C# Unity 3D Game Development in Unity 6» на Udemy.

## О проекте
Galaxy Strike — 3D аркадная игра с движением по рельсам, в которой игрок управляет кораблём, уклоняется и стреляет по врагам.

Проект позволил изучить:
- работу с 3D‑пространством и Terrain
- Unity Timeline
- корутины (Coroutines), массивы и циклы
- основы игрового цикла и механик на C#

## Реализовано
- Движение игрока по заранее заданному пути (rail logic)
- Стрельба и управление оружием
- Спавн врагов
- Простые эффекты попаданий и UI

## Технологии
- Unity 6
- C#
